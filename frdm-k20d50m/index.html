FRDM-K20D50M

The FRDM-K20D50M is the first Kinetis K-series offering in Freescale’s series of “Freedom” boards.

The specs look very similar to the FRDM-KL25Z, except that this is a Kinetis K20 USB part – a Cortex-M4 processor instead of a Cortex-M0+.

Freescale’s site claims a cost of USD18.00.

Here are some highlights:

  • MK20DX128VLH5 MCU – 50 MHz Cortex M4 processor core
  • 128 KiB Flash, 32 KiB FlexNVM, 16 KiB SRAM, USB OTG (full speed), LQFP 64
  • Capacitive touch sensor
  • MMA8451Q accelerometer
  • Ambient light sensor
  • RGB LED
  • CMSIS-DAP compatible
  • Header pin layout compatible with Arduino shields (but 3.3v only!)
